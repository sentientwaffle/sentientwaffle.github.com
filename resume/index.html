Stratus Editor

Stratus Editor is a web based programmer's editor with bundles, plugins (written in JavaScript or CoffeeScript), split screen, autocomplete, a file tree, and (soon!) Git integration.

Fractus is the standalone editor component of Stratus Editor.

Stratus-color is the syntax highlighter component of Stratus. It can also be used standalone via CLI or Node.js to highlight static code.

Tech used:

  • Node.js
  • CoffeeScript
  • jQuery
  • Underscore.js
  • Express
  • Socket.io
